(6) 修復 collation version mismatch
把 <DB> 換成你的資料庫名、<container> 換成容器名即可：

# 先重建整個資料庫的文字索引（並行、不會長時間阻塞寫入）
docker exec -it <container> psql -U postgres -d <DB> -v ON_ERROR_STOP=1 \
  -c "REINDEX (VERBOSE, CONCURRENTLY) DATABASE <DB>;"

# 再刷新資料庫的 collation 版本記錄（PG16 支援整庫刷新）
docker exec -it <container> psql -U postgres -d <DB> -v ON_ERROR_STOP=1 \
  -c "ALTER DATABASE <DB> REFRESH COLLATION VERSION;"


建議先確認真的有不一致：

SELECT n.nspname, c.collname, c.collversion AS recorded,
       pg_collation_actual_version(c.oid) AS actual
FROM pg_collation c
JOIN pg_namespace n ON n.oid = c.collnamespace
WHERE c.collversion IS DISTINCT FROM pg_collation_actual_version(c.oid);
